# Lazarus Protocol — Quick Start

## 5-minute setup

1. Clone and install: `pip install -r requirements.txt`
2. Run setup wizard: `python -m lazarus init`
3. Start agent: `python -m lazarus agent start`
4. Check status: `python -m lazarus status`
5. Open dashboard: `http://localhost:7777`

## Daily check-in
`python -m lazarus ping`

## Emergency freeze
`python -m lazarus freeze --days 30`
